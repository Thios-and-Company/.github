## What we're building

Architecture is locked behind architects, permits, and six-figure budgets. We think it shouldn't be.

Thios designs **modular structures you can actually build yourself** — a sauna, a workshop, a greenhouse, a focus room — each one engineered to fit an **8.5 × 8.5 ft footprint** (about one parking space) and assemble from materials you can buy at a normal hardware store. No specialty foundation, no crew, no architect on retainer.

Pick a structure, customize it in 3D, get a handbook with every cut, part, and step. Then go build it.
